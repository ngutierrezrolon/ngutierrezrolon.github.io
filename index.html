Blog Entry 1: Making a website

Making a website is very useful and can be very simple. This website is my personal portfolio and was made using HTML, CSS and Javascript.

HTML

HTML (HyperText Markup Language) is the part responsible for the structure of the content in the webpage. The subsections, headers, footers, hyperlinks, and main body of text are all done in this language.

CSS

CSS (Cascading Style Sheets) is the part responsible of all the main visuals, including fonts colors (in hexcode), spacing, etc.

JavaScript

Blog Entry 2: What is a "good enough" shuffle?

Random shuffles, randomness, card shuffling algorithms and more. If you like a good game of cards, whether that be (game1), game2 or game3, most card games start with a shuffle. As the game starts and the people around the table look at the cards, they groan and complain “who shuffled these?”. I want to talk about this conundrum of the shuffle, what the meaning of the word “random” even is, and explore how its practically applied and computationally replicated.

1- The Shuffle

There's several standard ways to shuffle. Casino shuffle. Overhand shuffle. Not all shuffles are mixing the same amount of cards, if you think about it. The basic overhand shuffle is removing a sector of cards from the deck, merging the rest of the deck and then adding them back on top. In this case a sector of size n is simply being displaced, but that sector will retain its original order entirely. How many “shuffles” exist? 52!
